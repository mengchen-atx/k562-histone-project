Final Project

Predicting K562 Gene Expression from Histone Modification Features

This project tested whether promoter and enhancer histone signals can explain gene expression in K562 cells. We first modeled continuous TPM values, found weak predictive power, then reframed the problem as binary and four-class classification.

Question Data Workflow TPM Regression Diagnosis Binary Classification Four-Class Classification Conclusion

Project Question

Biological Goal

We wanted to explain RNA expression levels using histone modification features in K562 cells. The central idea was that active and repressive chromatin marks around each gene might carry enough information to predict expression.

Modeling Goal

We first treated expression as a regression problem using log2(TPM + 1). Because regression performance remained poor, we also evaluated classification tasks: high vs low expression, and four expression groups.

Histone Marks Used

Mark Type Biological Meaning Where Applied
H3K4me3 Active Tri-methylation of histone H3 lysine 4; strongly marks active promoters and transcription start sites Promoter bins (TSS ±2 kb)
H3K27ac Active Acetylation of histone H3 lysine 27; marks active promoters and active enhancers Promoter bins + enhancer summaries (TSS ±50 kb)
H3K27me3 Repressive Tri-methylation of histone H3 lysine 27; deposited by Polycomb complexes; marks facultatively silenced genes that can be reactivated Promoter bins (TSS ±2 kb)
H3K9me3 Repressive Tri-methylation of histone H3 lysine 9; marks constitutive heterochromatin and permanently silenced regions Promoter bins (TSS ±2 kb)

Note: H3K36me3, a gene-body elongation mark that is among the strongest predictors of expression level in the literature, was not available in our dataset. Its absence is a likely contributor to the limited regression performance observed (see Conclusion).

Data and Features

Cell Line

K562 only. All histone ChIP-seq and RNA-seq values were restricted to the same cell line. K562 is a chronic myelogenous leukemia (CML) cell line; its epigenome is substantially altered relative to normal cells, so findings may not generalize directly to non-cancer contexts.

Target

RNA_TPM, transformed as log2(TPM + 1) for regression.

Feature Set

82 total features: 80 promoter-bin features plus 2 enhancer H3K27ac summary features.

Feature Group Description Count
Promoter bins TSS ±2 kb, split into 20 bins, 4 marks per bin 80
Enhancer summaries H3K27ac enhancer max and enhancer sum within TSS ±50 kb excluding promoter 2
Total All numerical predictors used in modeling 82

Analysis Workflow

1. Feature Matrix Preparation

We cleaned the merged matrix, restricted the analysis to protein-coding genes, removed rows with missing values, and excluded genes with RNA_TPM = 0 and all histone features equal to 0. We then created filtered subsets using TPM thresholds > 0, > 0.1, > 1, and > 5 for sensitivity analysis.

2. Continuous Regression

We tested Ridge regression, Random Forest, HistGradientBoosting, and MLP on continuous TPM prediction.

3. Model Diagnosis

We inspected prediction scatter plots and feature distributions between highly and weakly expressed genes to check whether the signal was missing, weak, or misaligned.

4. Reframing the Task

Because regression remained weak, we reframed the problem into binary classification and then four-class classification.

TPM Regression Results

Regression was run on the protein-coding, TPM > 0.1 subset. The goal was to predict continuous log2(TPM + 1). All four models performed poorly, with near-zero or negative explanatory power.

Why Regression Performed Poorly

Scatter Plot Pattern

Predicted values were compressed into a narrow range rather than following the ideal diagonal line. This suggested that the models were mostly learning an average expression level instead of a strong gene-specific signal.

  • Predictions clustered around a small interval.
  • R² stayed near zero or below zero across models.
  • This pointed to weak feature signal rather than a single bad model choice.

Row-Level Diagnosis

We compared top 200 and bottom 200 expressed genes. H3K4me3_bin10 showed almost no separation, while H3K27ac_enhancer_sum had some separation but still with large overlap.

  • Some features have signal, but it is weak.
  • The enhancer summary carried more signal than at least some promoter bins.
  • The full 82-feature set is not strong enough for accurate TPM regression.

Binary Classification: High vs Low Expression

We then reframed the task as a binary problem. Genes in the highest 25% of expression were labeled as high, genes in the lowest 25% were labeled as low, and the middle 50% were excluded.

Interpretation

Binary classification clearly performed better than continuous regression. The best model, Random Forest, reached a ROC AUC of 0.7065, showing that the histone features contain enough signal to separate broad expression states even when they cannot predict exact TPM values accurately.

Four-Class Classification: Expression Quartiles

Finally, we divided all genes into four equal expression groups based on ranked log2(TPM + 1). This tested whether the same features could distinguish finer expression categories.

Interpretation

Four-class classification was harder. Accuracy stayed around 0.32 to 0.33, only modestly above random guessing for four classes. This means the current feature set can separate broad expression states better than it can distinguish fine-grained expression levels.

Conclusion and Reflection

Main Findings

  • Continuous TPM regression performed poorly for all tested models.
  • Some histone features, especially enhancer-related H3K27ac summaries, still contained usable signal.
  • Binary classification produced the clearest improvement, with Random Forest giving the strongest result.
  • Four-class classification remained difficult, suggesting the features are not rich enough for fine expression ranking.

What We Learned

  • Model choice mattered less than feature quality.
  • Reframing the biological question changed the outcome substantially.
  • For this project, chromatin features were more useful for separating broad expression states than predicting exact TPM.
  • Future improvements should focus on stronger feature engineering, such as gene-body marks, chromatin accessibility, or richer enhancer summaries.

Where Does the Binary Classification Signal Come From?

To probe how much of the binary classification signal was driven by distinguishing completely silent genes from active ones, we re-ran all four classifiers after restricting the dataset to genes with TPM > 0.1 — the same filtered set used in the regression experiments.

ROC AUC dropped substantially across all models, falling close to random-chance levels:

Model Full gene set AUC
(includes silent genes)		 TPM > 0.1 only AUC Drop
Random Forest 0.7065 0.5934 −0.113
MLP 0.6852 0.5674 −0.118
Logistic Regression 0.6796 0.5530 −0.127
HistGradientBoosting 0.6742 0.5735 −0.101

This result indicates that most of the original binary classification signal reflected the distinction between silenced genes (repressive marks such as H3K27me3 and H3K9me3 dominate) and active genes, rather than fine-grained differences in expression level within the expressed population. Within expressed genes, the four-mark promoter-region feature set provides only marginal predictive power — consistent with literature showing that H3K36me3, a gene-body elongation mark absent from our features, accounts for the majority of quantitative expression prediction power (Karlic et al., 2010, PNAS).
