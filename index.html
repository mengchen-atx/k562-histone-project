Final Project

Predicting K562 Gene Expression from Histone Modification Features

Group Members: Meng Chen, Weiyang Hong, Adesola Johnson

This project tested whether promoter and enhancer histone signals can explain gene expression in K562 cells, using six histone marks: H3K4me3, H3K27ac, H3K27me3, H3K9me3, H3K36me3, and H3K4me1. We modeled continuous TPM values, found limited regression performance, then reframed the problem as binary and four-class classification.

Project Question

Research Gap

The epigenetic regulation of gene expression is a widely studied topic, with researchers often gathering new insight into how chromatin remodeling, DNA methylation/demethylation, and histone modification factors control gene activity. These epigenetic changes not only affect DNA accessibility but also contribute to tissue development, disease progression, and cellular reprogramming after injury (Bernstein et al., 2007). The intersection of biology, statistics, and machine learning has led to advancements in our understanding of this complex cellular phenomenon and its representation in the mammalian genome. However, only a few machine learning models accurately depict how these epigenetic mechanisms, especially histone modifications, determine gene expression–creating a knowledge gap that this project aims to address.

Histone Marks Used

Mark Type Biological Meaning Where Applied
H3K4me3 Active Tri-methylation of histone H3 lysine 4; strongly marks active promoters and transcription start sites Promoter bins (TSS ±2 kb)
H3K27ac Active Acetylation of histone H3 lysine 27; marks active promoters and active enhancers Promoter bins + enhancer summaries (TSS ±50 kb)
H3K27me3 Repressive Tri-methylation of histone H3 lysine 27; deposited by Polycomb complexes; marks facultatively silenced genes that can be reactivated Promoter bins (TSS ±2 kb)
H3K9me3 Repressive Tri-methylation of histone H3 lysine 9; marks constitutive heterochromatin and permanently silenced regions Promoter bins (TSS ±2 kb)
H3K36me3 Active Tri-methylation of histone H3 lysine 36; deposited co-transcriptionally across gene bodies; among the strongest quantitative predictors of expression level Promoter bins (TSS ±2 kb)
H3K4me1 Active / Poised Mono-methylation of histone H3 lysine 4; marks active and poised enhancers; enriched at distal regulatory elements rather than core promoters Promoter bins (TSS ±2 kb)

Data and Features

Cell Line

K562 only. All histone ChIP-seq and RNA-seq values were restricted to the same cell line. K562 is a chronic myelogenous leukemia (CML) cell line; its epigenome is substantially altered relative to normal cells, so findings may not generalize directly to non-cancer contexts.

Target

RNA_TPM, transformed as log2(TPM + 1) for regression.

Feature Set

122 total features: 120 promoter-bin features plus 2 enhancer H3K27ac summary features.

Feature Group Description Count
Promoter bins TSS ±2 kb, split into 20 bins, 6 marks per bin 120
Enhancer summaries H3K27ac enhancer max and enhancer sum within TSS ±50 kb excluding promoter 2
Total All numerical predictors used in modeling 122

Analysis Workflow

1. Feature Matrix Preparation

We applied the following filters to the merged feature matrix in order:

  • Protein-coding genes only — non-coding RNA genes, pseudogenes, and other biotypes were excluded using NCBI gene annotation.
  • Remove missing values — rows with any NaN in feature or target columns were dropped.
  • Exclude double-zero rows — genes with both RNA_TPM = 0 and all histone features equal to 0 were removed, as they carry no signal.
  • TPM threshold subsets — additional filtered versions at TPM > 0, > 0.1, > 1, and > 5 were created for sensitivity analysis.

2. Continuous Regression

We tested Ridge regression, Random Forest, HistGradientBoosting, and MLP on continuous TPM prediction.

3. Model Diagnosis

We inspected prediction scatter plots and feature distributions between highly and weakly expressed genes to check whether the signal was missing, weak, or misaligned.

4. Reframing the Task

Because regression remained weak, we reframed the problem into binary classification and then four-class classification.

TPM Regression Results

Regression was run on the protein-coding, TPM > 0.1 subset (11,436 genes, 122 features). The goal was to predict continuous log2(TPM + 1). Adding H3K36me3 and H3K4me1 improved R² and Pearson r substantially compared to the four-mark dataset, though performance remains modest — consistent with the difficulty of fine-grained TPM prediction from promoter bins alone.

R²
Variance in log2(TPM+1) explained by the model. Range −∞ to 1; ideal = 1. Values ≤ 0 indicate the model is no better than predicting the mean.
Pearson r
Linear correlation between predicted and actual values. Range −1 to 1; ideal = 1.
RMSE
√(mean squared error), in log2(TPM+1) units. Penalises large errors heavily. Ideal = 0.
MAE
Mean absolute error, in log2(TPM+1) units. More robust to outliers than RMSE. Ideal = 0.

Binary Classification: High vs Low Expression

We then reframed the task as a binary problem. Genes in the highest 25% of expression were labeled as high, genes in the lowest 25% were labeled as low, and the middle 50% were excluded.

Accuracy
Fraction of genes correctly classified: (TP + TN) / total. Ideal = 1.
Precision
Of genes predicted high-expression, fraction that truly are: TP / (TP + FP). Ideal = 1.
Recall
Of all truly high-expression genes, fraction correctly identified: TP / (TP + FN). Ideal = 1.
F1
Harmonic mean of Precision and Recall. Balances both in one number. Ideal = 1.
ROC AUC
Probability the model ranks a high-expression gene above a low-expression gene. Threshold-independent. Range 0.5 (random) to 1; ideal = 1.

Interpretation

Binary classification performed substantially better than continuous regression. With the expanded six-mark feature set, the best models (Random Forest and HistGradientBoosting) reached a ROC AUC above 0.85 — a meaningful jump from the four-mark results — confirming that H3K36me3 and H3K4me1 add discriminative power for separating broad expression states.

Four-Class Classification: Expression Quartiles

Finally, we divided all genes into four equal expression groups based on ranked log2(TPM + 1). This tested whether the same features could distinguish finer expression categories.

Interpretation

Four-class classification was harder. Accuracy ranged from 0.38 to 0.41 — an improvement over the four-mark results (~0.32–0.33) but still modest above the random-chance baseline of 0.25. This confirms that while the expanded feature set adds signal, the current promoter-bin representation is not rich enough for fine-grained expression ranking across all four quartiles.

Conclusion and Reflection

Main Findings

  • Adding H3K36me3 and H3K4me1 substantially improved all tasks relative to the four-mark baseline.
  • Continuous TPM regression remained modest (best R² ≈ 0.10, Pearson r ≈ 0.32) even with six marks.
  • Binary classification benefited most, with ROC AUC rising from ~0.71 to ~0.85.
  • Four-class accuracy improved from ~0.33 to ~0.41, but remains limited for fine-grained expression ranking.

What We Learned

  • Feature quality dominated model choice — adding two marks mattered more than tuning any single algorithm.
  • H3K36me3, a gene-body elongation mark, appears to drive the largest share of the performance gain.
  • Reframing as binary classification yields the most interpretable results from the current feature set.
  • Further gains likely require gene-body coverage, chromatin accessibility (ATAC-seq), or richer long-range enhancer features.
