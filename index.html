Final Project

Predicting K562 Gene Expression from Histone Modification Features

This project tested whether promoter and enhancer histone signals can explain gene expression in K562 cells, using six histone marks: H3K4me3, H3K27ac, H3K27me3, H3K9me3, H3K36me3, and H3K4me1. We modeled continuous TPM values, found limited regression performance, then reframed the problem as binary and four-class classification.

Project Question

Research Gap

The epigenetic regulation of gene expression is a widely studied topic, with researchers often gathering new insight into how chromatin remodeling, DNA methylation/demethylation, and histone modification factors control gene activity. These epigenetic changes not only affect DNA accessibility but also contribute to tissue development, disease progression, and cellular reprogramming after injury (Bernstein et al., 2007). The intersection of biology, statistics, and machine learning has led to advancements in our understanding of this complex cellular phenomenon and its representation in the mammalian genome. However, only a few machine learning models accurately depict how these epigenetic mechanisms, especially histone modifications, determine gene expression–creating a knowledge gap that this project aims to address.

Data and Features

Cell Line

K562 only. All histone ChIP-seq and RNA-seq values were restricted to the same cell line. K562 is a chronic myelogenous leukemia (CML) cell line; its epigenome is substantially altered relative to normal cells, so findings may not generalize directly to non-cancer contexts.

Target

RNA_TPM, transformed as log2(TPM + 1) for regression.

Feature Set

122 total features: 120 promoter-bin features plus 2 enhancer H3K27ac summary features.

Feature Group Description Count
Promoter bins TSS ±2 kb, split into 20 bins, 6 marks per bin 120
Enhancer summaries H3K27ac enhancer max and enhancer sum within TSS ±50 kb excluding promoter 2

Analysis Workflow

1. Feature Matrix Preparation

We applied the following filters to the merged feature matrix in order:

  • Protein-coding genes only — non-coding RNA genes, pseudogenes, and other biotypes were excluded using NCBI gene annotation.
  • Remove missing values — rows with any NaN in feature or target columns were dropped.
  • Exclude double-zero rows — genes with both RNA_TPM = 0 and all histone features equal to 0 were removed, as they carry no signal.
  • TPM threshold subsets — additional filtered versions at TPM > 0, > 0.1, > 1, and > 5 were created for sensitivity analysis.

2. Continuous Regression

We tested Ridge regression, Random Forest, HistGradientBoosting, and MLP on continuous TPM prediction.

3. Model Diagnosis

We inspected prediction scatter plots and feature distributions between highly and weakly expressed genes to check whether the signal was missing, weak, or misaligned.

4. Reframing the Task

Because regression remained weak, we reframed the problem into binary classification and then four-class classification.

TPM Regression Results

Regression was run on the protein-coding, TPM > 0.1 subset (11,436 genes, 122 features). The goal was to predict continuous log2(TPM + 1). Adding H3K36me3 and H3K4me1 improved R² and Pearson r substantially compared to the four-mark dataset, though performance remains modest — consistent with the difficulty of fine-grained TPM prediction from promoter bins alone.

Model R² Pearson r RMSE MAE
Ridge Regression 0.0912 0.3047 1.6446 1.2998
Random Forest 0.1005 0.3196 1.6361 1.2984
HistGradientBoosting 0.0885 0.3106 1.6470 1.3019
MLP 0.0585 0.2773 1.6739 1.3159
R² (Coefficient of Determination)
Proportion of variance in log2(TPM + 1) explained by the model. R² = 1 − SSres / SStot, where SSres is the sum of squared residuals and SStot is the total variance around the mean. Range: −∞ to 1; higher is better. R² ≤ 0 means the model is no better than predicting the mean.
Pearson r (Pearson Correlation Coefficient)
Linear correlation between actual and predicted log2(TPM + 1) values across the test set. r = cov(ŷ, y) / (σŷ · σy). Range: −1 to 1; higher magnitude is better. Unlike R², it is not sensitive to systematic bias in scale.
RMSE (Root Mean Squared Error)
Square root of the average squared difference between predicted and actual values: RMSE = √(Σ(ŷi − yi)² / n). Units match the target (log2(TPM + 1)); lower is better. Penalises large errors more heavily than MAE.
MAE (Mean Absolute Error)
Average absolute difference between predicted and actual values: MAE = Σ|ŷi − yi| / n. Units match the target; lower is better. More robust to outliers than RMSE.

Why Regression Performed Poorly

Scatter Plot Pattern

Predicted values were compressed into a narrow range rather than following the ideal diagonal line. This suggested that the models were mostly learning an average expression level instead of a strong gene-specific signal.

  • Predictions clustered around a small interval.
  • R² stayed near zero or below zero across models.
  • This pointed to weak feature signal rather than a single bad model choice.

Row-Level Diagnosis

We compared top 200 and bottom 200 expressed genes. H3K4me3_bin10 showed almost no separation, while H3K27ac_enhancer_sum had some separation but still with large overlap.

  • Some features have signal, but it is weak.
  • The enhancer summary carried more signal than at least some promoter bins.
  • Even the expanded 122-feature set is not sufficient for accurate fine-grained TPM regression.

Binary Classification: High vs Low Expression

We then reframed the task as a binary problem. Genes in the highest 25% of expression were labeled as high, genes in the lowest 25% were labeled as low, and the middle 50% were excluded.

Random Forest 0.7753 0.7955 0.7209 0.7564 0.8503
HistGradientBoosting 0.7722 0.8042 0.6994 0.7481 0.8515

Accuracy
Fraction of test genes correctly classified: (TP + TN) / (TP + TN + FP + FN). Intuitive but can be misleading when classes are imbalanced. Here classes are balanced by construction (equal-sized quartile groups).
Precision
Of all genes predicted as "high expression", what fraction actually are: Precision = TP / (TP + FP). High precision means few false positives — the model is conservative about calling a gene highly expressed.
Recall (Sensitivity)
Of all truly high-expression genes, what fraction did the model correctly identify: Recall = TP / (TP + FN). High recall means few missed high-expression genes. Precision and Recall trade off against each other.
F1 Score
Harmonic mean of Precision and Recall: F1 = 2 · (Precision · Recall) / (Precision + Recall). Balances both concerns in a single number. Range: 0 to 1; higher is better.
ROC AUC (Area Under the Receiver Operating Characteristic Curve)
Probability that the model ranks a randomly chosen high-expression gene above a randomly chosen low-expression gene. The ROC curve plots True Positive Rate vs False Positive Rate at all classification thresholds; AUC summarises the full curve. Range: 0.5 (random) to 1.0 (perfect); higher is better. Unlike Accuracy, AUC is threshold-independent.

Interpretation

Binary classification performed substantially better than continuous regression. With the expanded six-mark feature set, the best models (Random Forest and HistGradientBoosting) reached a ROC AUC above 0.85 — a meaningful jump from the four-mark results — confirming that H3K36me3 and H3K4me1 add discriminative power for separating broad expression states.

Four-Class Classification: Expression Quartiles

Finally, we divided all genes into four equal expression groups based on ranked log2(TPM + 1). This tested whether the same features could distinguish finer expression categories.

Model Accuracy Macro Precision Macro Recall Macro F1
Logistic Regression 0.3837 0.3744 0.3836 0.3741
Random Forest 0.4063 0.3993 0.4063 0.3994
HistGradientBoosting 0.3829 0.3775 0.3828 0.3766
MLP 0.3912 0.3793 0.3912 0.3797
Accuracy
Fraction of test genes assigned to the correct quartile class (Q1–Q4) out of all predictions. Random-chance baseline for four equal classes is 0.25.
Macro Precision
Precision computed independently for each of the four classes, then averaged with equal weight: Macro Precision = (1/4) Σ TPk / (TPk + FPk). "Macro" means each class contributes equally regardless of size, so performance on the rare classes is not diluted.
Macro Recall
Recall computed per class and averaged: Macro Recall = (1/4) Σ TPk / (TPk + FNk). Captures whether the model finds each expression quartile, not just the most frequent one.
Macro F1
Harmonic mean of Macro Precision and Macro Recall, averaged across the four classes. Summarises overall balance between precision and recall for the multi-class problem. Range: 0 to 1; higher is better.

Interpretation

Four-class classification was harder. Accuracy ranged from 0.38 to 0.41 — an improvement over the four-mark results (~0.32–0.33) but still modest above the random-chance baseline of 0.25. This confirms that while the expanded feature set adds signal, the current promoter-bin representation is not rich enough for fine-grained expression ranking across all four quartiles.

Conclusion and Reflection

Main Findings

  • Adding H3K36me3 and H3K4me1 substantially improved all tasks relative to the four-mark baseline.
  • Continuous TPM regression remained modest (best R² ≈ 0.10, Pearson r ≈ 0.32) even with six marks.
  • Binary classification benefited most, with ROC AUC rising from ~0.71 to ~0.85.
  • Four-class accuracy improved from ~0.33 to ~0.41, but remains limited for fine-grained expression ranking.

What We Learned

  • Feature quality dominated model choice — adding two marks mattered more than tuning any single algorithm.
  • H3K36me3, a gene-body elongation mark, appears to drive the largest share of the performance gain.
  • Reframing as binary classification yields the most interpretable results from the current feature set.
  • Further gains likely require gene-body coverage, chromatin accessibility (ATAC-seq), or richer long-range enhancer features.

Impact of Adding H3K36me3 and H3K4me1

The V3 dataset expanded the feature set from four marks to six by adding H3K36me3 (a gene-body elongation mark) and H3K4me1 (an enhancer-associated mark). Across all tasks, performance improved noticeably:

Task / Metric 4-mark (V2) 6-mark (V3) Change
Regression best R² −0.0098 0.1005 +0.11
Regression best Pearson r 0.1066 0.3196 +0.21
Binary classification best ROC AUC 0.7065 0.8515 +0.145
Four-class best accuracy 0.3309 0.4063 +0.075

H3K36me3, which is deposited co-transcriptionally across actively transcribed gene bodies, is among the strongest single-mark predictors of expression level in the literature (Karlic et al., 2010, PNAS). Its inclusion here accounts for a large share of the observed gain. H3K4me1 contributes additional signal by capturing enhancer activity near promoters. Despite these improvements, fine-grained TPM regression and four-class ranking remain difficult, suggesting that the promoter-bin representation still misses regulatory information captured by gene-body coverage, chromatin accessibility, or richer long-range enhancer interactions.
